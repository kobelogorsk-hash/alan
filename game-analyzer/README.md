# Brawl Stars Progress Analyzer

Программа для анализа прогресса в Brawl Stars, выявления плюсов и минусов игровой производительности.

## Возможности

- 📊 Отслеживание статистики по Brawl Stars
- 📈 Анализ прогресса трофеев во времени
- ✅ Выявление сильных сторон (высокий K/D, урон, лечение)
- ❌ Определение слабых мест (низкая точность, частые смерти)
- 🎯 Рекомендации по улучшению
- 🎨 Современный GUI интерфейс
- 💾 Автосохранение данных

## Интерфейс GUI

Приложение имеет 3 вкладки:

1. **Добавить игру** - форма для ввода результатов матча
   - Боец, режим, место
   - Изменение трофеев
   - Убийства, смерти, урон, лечение
   - Использование супер-атаки

2. **Статистика** - аналитика и графики
   - График прогресса трофеев
   - Общая статистика (K/D, средние значения)
   - Плюсы и минусы вашей игры
   - Общий рейтинг 0-100

3. **История** - таблица всех матчей
   - Последние 20 игр
   - Возможность удаления записей
   - Сортировка по параметрам

## Пример использования (консоль)

```python
from games.brawl_stars import BrawlStarsGame

bs = BrawlStarsGame()

# Создание сессии
session = bs.create_session({
    "brawler": "Шелли",
    "mode": "Brawl Ball",
    "rank": 1,
    "trophy_change": 8,
    "kills": 5,
    "deaths": 2,
    "damage": 3500,
    "healing": 0,
    "super_used": True
})

# Анализ
analysis = bs.analyze_performance([session])
print(f"Рейтинг: {analysis['overall_score']}/100")
print("Плюсы:", analysis['strengths'])
print("Минусы:", analysis['weaknesses'])
```

## Метрики Brawl Stars

### Основные:
- **Трофеи** - прогресс в рангах
- **K/D Ratio** - соотношение убийств к смертям
- **Урон** - средний урон за матч
- **Лечение** - поддержка команды

### Режим-специфичные:
- **Brawl Ball** - голы, перехваты
- **Gem Grab** - собранные кристаллы
- **Showdown** - выживание, место

## Советы по улучшению

Программа автоматически генерирует рекомендации на основе:
- Частоты использования разных бойцов
- Эффективности в различных режимах
- Динамики изменения трофеев
- Сравнения со средними показателями
